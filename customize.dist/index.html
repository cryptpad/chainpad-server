Chainpad Server

Welcome to your Chainpad Server instance! You'll find everything you need to get started writing software for Collaborative Real Time Editing.

  • requirejs: Javascript module loader
  • chainpad: the Realtime Collaborative Editor algorithm
  • jquery: the "Write Less, Do More" Javascript library
  • hyperjson: for converting a DOM subtree into JSON for serialization
  • json.sortify: for browser-independent JSON serialization
  • textpatcher: for diffing on text
  • tweetnacl: for clientside encryption

To get started, check out XWiki Labs' Realtime Handbook.
